# 💡 Best Practices

* Keep your GitHub token secure.
* Store API keys in the `.env` file.
* Use Ollama for offline analysis.
* Use OpenAI for advanced reasoning tasks.
* Review AI suggestions before applying them.
* Export important reports for documentation.
